ON GRID SOLAR SYSTEM

What is Solar On Grid?

The On grid solar system is connected with a utility grid and this system would work only if the grid is available. In case of a power cut, the system will not work and hence the On-grid system is also termed a Grid-tied system.

How it works?

A grid is required since the inverter needs to be provided a reference voltage and the inverter needs to sync with the grid in order to export energy back into the grid.

OFF GRID SOLAR SYSTEM

Why Off Grid?

Off-grid solar systems are independent power solutions that operate without relying on the utility grid. Ideal for remote locations or areas prone to power outages.
